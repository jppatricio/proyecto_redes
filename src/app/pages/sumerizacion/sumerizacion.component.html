Superneteo o sumarización de rutas

El superneteo es una técnica que permite agrupar múltiples rutas IP contiguas en una única ruta.

Ventajas
  • Reduce el volumen de información de enrutamiento publicado.
  • Reduce la complejidad de las tablas de enrutamiento, reduciendo la cantidad de rutas.
  • Aumenta la estabilidad de las tablas de ruteo.
  • Reduce los requirimientos de procesamiento ya que minimiza los procedimientos de actualización de rutas y se reduce la cantidad de rutas a evaluar.

Para aplicar este método la direcciones IP deben ser contiguas
Esta técnica es utilizada por los ISP (Proveedores de servicios de Internet)

Pasos a seguir
  1. Verificar que las direcciones IP de las redes sean contiguas.
  2. Posteriormente se ordenan de las direcciones IP de menor a mayor.
  3. Cada dirección se convierte a binario.
  4. Se toma la cadena de bits que es idéntica en todas las direcciones.
  5. Interpretar esta cadena como la dirección IP de la red y su respectiva máscara.
